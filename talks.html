The Many Faces of Optimal Weak-to-Strong Learning

In this presentation we will study the concept of weak-to-strong learners, with an emphasis on boosting. In particular, we will focus on a paper by Mikael Møller Høgsgaard, Kasper Green Larsen, Markus Engelund Mathiasen which presents a surprisingly simple boosting algorithm that achieves optimal sample complexity. Previous work has shown a theoretical lower bound for sample complexity. Sample optimal boosting algorithms have only recently been developed, but we will show that this new algorithm has faster runtime while being simpler to implement.

IP=PSPACE

Interactive Proofs (IP) represent a powerful computational model where a probabilistic verifier can efficiently validate complex computational claims through interaction with a prover. In this presentation we will study the breakthrough result in complexity theory IP = PSPACE. We will show how through randomization + interaction, efficient interactive proofs are able to achieve this extremely powerful result. In particular, we will study Shamir’s (1992) proof of this concept presenting an IP protocol for quantified boolean formulas, as well as a simplification of this procedure presented by Shen (1992).
